# Supenli.ai — High-Fidelity Rebuild

A pixel-perfect, fully functional rebuild of [supenli.ai](https://www.supenli.ai) — the AI-powered viral content generator.

**Live Demo:** Deployed automatically on Vercel.

## Features
- Exact copy and structure from original landing page
- Beautiful modern design with violet accents (Tailwind + Framer Motion)
- Fully interactive demo modal: Generate 5 smart viral variations in real-time (client-side AI simulation)
- Platform selection, copy to clipboard, fake infographic generation
- Comparison table, testimonials, metrics, pricing with fake checkout
- FAQ accordion, smooth scrolling, responsive on all devices
- Auth simulation (sign up / sign in modals)
- Toast notifications

## Tech Stack
- Next.js 15 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion (animations & modals)
- Lucide React (icons)
- Custom toast system

## Getting Started (Local)
```bash
npm install
npm run dev
```

## Deployment
This repo is connected to Vercel for instant deployments on every push.
